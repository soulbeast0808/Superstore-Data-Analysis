## Superstore Sales Data Analysis


## Project Description
This project analyzes the Superstore dataset to understand sales trends,
profitability, and business insights using Python.
  
   
## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
